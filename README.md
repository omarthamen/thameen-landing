# ثَمين — صفحة الهبوط (Landing Page)

صفحة واحدة متجاوبة (جوال + كمبيوتر)، عربية RTL، بهوية ثَمين. تعرض فيديو رئيسي طويل ومعرض صور، مع أزرار توّدي لـ thameen.shop.

## التشغيل
انقر مرتين على `index.html` لفتحها في المتصفح — لا تحتاج أي تثبيت.
(لو تبي تشغّلها كسيرفر محلي: `cd ~/thameen-landing && python3 -m http.server 8000` ثم افتح <http://localhost:8000>.)

## كيف تضيف الفيديو الطويل
1. سمِّ ملف فيديوك `main.mp4`.
2. ضعه في `assets/video/` (يستبدل الموجود).
3. (اختياري) ضع صورة غلاف للفيديو باسم `poster.jpg` في نفس المجلد، وغيّر `poster.svg` إلى `poster.jpg` في `index.html`.

> النوع المدعوم: MP4 (H.264) — الأفضل للتوافق على كل الأجهزة.

## كيف تضيف صورك
1. انسخ صورك (jpg / png / webp) إلى `assets/images/`.
2. احذف ملفات `placeholder-*.svg` لو ما تبيها.
3. انقر مرتين على **`update-gallery.command`** — يحدّث المعرض تلقائيًا.
4. حدّث الصفحة في المتصفح.

الصور تُرتّب أبجديًا حسب الاسم، فاستخدم أرقامًا (`01.jpg`, `02.jpg`...) للتحكم بالترتيب.

## تعديل الألوان والخطوط
كل الهوية في أعلى `styles.css` داخل `:root` (مطابقة لـ `brand.ts`). غيّرها من هناك.

## النشر على الإنترنت
ارفع كامل مجلد `thameen-landing/` على أي استضافة ثابتة (Netlify / Cloudflare Pages / GitHub Pages / استضافة المتجر). لا يوجد build — ملفات ثابتة فقط.

## الملفات
- `index.html` — الصفحة
- `styles.css` — التصميم والهوية
- `main.js` — المعرض و Lightbox
- `gallery.js` — قائمة الصور (مولّدة تلقائيًا، لا تعدّلها يدويًا)
- `update-gallery.command` — يحدّث `gallery.js` من مجلد الصور
- `assets/video/` — `main.mp4` + `poster.svg`
- `assets/images/` — صور المعرض
